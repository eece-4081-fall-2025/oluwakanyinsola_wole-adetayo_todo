# Oluwakanyinsola_Wole-Adetayo_todo

Simple Django todo app for course assignment.


## ✅ Epic 1 – Task Flow (MVP)

Implements all core Task Flow user stories:
- Create a new task with a title and description  
- Edit an existing task  
- Delete a task  
- Toggle a task as completed or uncompleted  
- Minimal interface to view and manage tasks  

This Epic was delivered using **Test-Driven Development (TDD)** with two full cycles of RED → GREEN commits.

---

## Run locally
1. python3 -m venv .venv
2. source .venv/bin/activate
3. pip install -r requirements.txt
4. python manage.py migrate
5. python manage.py runserver
6. open http://127.0.0.1:8000/
